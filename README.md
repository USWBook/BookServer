### 📌 1. 프로젝트명
**Subook -- 대학생을 위한 중고 교재 거래 플랫폼**


### 📚 2. 프로젝트 소개
> **Subook**는 대학생들이 전공서적과 교양서를 간편하게 사고팔 수 있도록 지원하는 중고책 거래 서비스입니다.
> 수강 신청, 캠퍼스 활동, 전공 관리 등 다양한 기능을 제공하여 학생과 교직원의 대학 생활을 쉽고 편리하게 만듭니다.

### 🚀 3. 주요 기능
- [ ] 회원가입 / 로그인
- [ ] 게시글 기능
- [ ] 채팅 기능
- [ ] 마이페이지
- [ ] 검색 / 필터 기능
- [ ] 관리자 기능
